About Me

Hai! Perkenalkan namaku I Gusti Ngurah Cipta Dwipajaya. Umurku 20 tahun, saya adalah seorang mahasiwa yang mengambil program studi Information System. Saat ini saya sedang mempelajari bidang programming dan berfokus pada web developer.

Pada awalnya saya tidak tahu tentang dunia programming. Tetapi, setelah berkuliah di ITB Stikom Bali saya mengetahui bidang ini karena memperoleh mata kuliah algoritma dan struktur data. Pada awalnya saya kurang tertarik dengan programming dan menganggap itu adalah hal yang sulit. Namun, setelah semester 4 saya kembali mulai mempelajari dunia programming dimulai dari mempelajari C++, Java, Html, Css dan Javascript.
